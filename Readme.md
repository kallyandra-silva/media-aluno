# 🎓 Projeto de Situação Escolar

## 🗒️ Sobre o projeto

Este projeto foi criado a partir de um fluxograma que define a lógica para calcular a situação escolar de um aluno. O código segue as etapas do fluxograma, recebendo as notas e a frequência, calculando a média e informando se o aluno está 🟢aprovado, em 🟡recuperação ou 🔴reprovado.  

## ⚙️ Funcionalidades

- Leitura do nome do aluno
- Leitura das notas N1 e N2
- Leitura da frequência
- Cálculo da média final
- Verificação da situação do aluno
 → Aprovado
 → Recuperação
 → Reprovado
- Exibição dos Resultados
